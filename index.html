CFMet Home

The CFmet site has been built to provide useful resources to aid meteorological technicians with their everyday duties. The website is still undergoing construction and some features may not be fully functional at this time. If you encounter issues, please contact the site administrator with details.

Currently Included:

  • The Links page provides a groing repository of urls for a wide range of meteorology resources including: satellites, radars, cameras, aviation tools, publications, research papers, journals, courses, and more.
  • The Tools page, for now, provides only a useful widget for calculating 909:RtDc codes, along with the calculation table and the associated Manobs reference. More tools will be added in the future.
