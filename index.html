About This Application: [HIDE]

Welcome and thank you for using the 909:RtDc Calculator.

This tool was originally created to aid technicians with the calculation of Rt and Dc - the start, stop and duration of precipitation - at weather stations still recording synoptic observations.

This tool is to be used as an aid only and it is highly recommended that technicians still calculate Rt and Dc codes manually using the table provided in MANOBS.

On that note: the program has undergone months of testing and has been in operation for roughly six months now without error.

If any errors, or bugs are found or you have ideas to help make the program better, please report them to the creator of the site using the form provided at the bottom of the site.
